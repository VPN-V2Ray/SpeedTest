## SpeedTest.sh#######

在Linux VPS上测试网络连接速度，使用全国三大运营商的测速节点进行全面测速。
包括中国大陆地区电信、联通、移动节点。

## 使用方法
```
bash <(curl -Lso- https://git.io/SpeedTest)
```
